# Infraestrutura-de-Redes-Senac-
Projeto integrador de Infraestrutura de Redes desenvolvido no Senac. Planejamento e implementação de rede corporativa com topologia documentada, configuração de dispositivos, serviços essenciais (DNS, DHCP, AD, Firewall) e documentação técnica para garantir conectividade, segurança e desempenho.
